# Dev Portfolio — Lucas Freitas

Portfolio pessoal desenvolvido do zero com HTML, CSS e JavaScript puro. O objetivo foi criar uma presença digital limpa, responsiva e com identidade visual consistente, inspirada no estilo de IDEs e terminais de desenvolvimento.

---

## Funcionalidades

- **Hero section** com animações de entrada encadeadas
- **Sobre mim** com lista de skills em formato de badge
- **Seção de projetos** com cards interativos e links para demo e repositório
- **Seção de contato** com links diretos para e-mail, GitHub e LinkedIn
- **Scroll reveal** via IntersectionObserver para animações ao rolar a página
- **Design totalmente responsivo** para mobile e desktop

---

## Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties, Animações)
- JavaScript (Vanilla)
- Google Fonts: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) + [Sora](https://fonts.google.com/specimen/Sora)

## Como rodar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/LucasFerreiradeFreitas/<nome-do-repo>.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd <nome-do-repo>
   ```

3. Abra o arquivo `index.html` no navegador — não há dependências ou build necessário.

> Dica: use a extensão **Live Server** do VS Code para recarregamento automático durante o desenvolvimento.

## Design system

O projeto utiliza um conjunto de variáveis CSS centralizadas:

| Variável      | Valor     | Uso                        |
|---------------|-----------|----------------------------|
| `--bg`        | `#0d1117` | Fundo principal            |
| `--surface`   | `#161b22` | Cards e painéis            |
| `--border`    | `#30363d` | Bordas e divisores         |
| `--text`      | `#e6edf3` | Texto principal            |
| `--muted`     | `#8b949e` | Texto secundário           |
| `--accent`    | `#58a6ff` | Destaques e links          |
| `--green`     | `#3fb950` | Labels, tags de tecnologia |
